# APB_Demo Monthly Spending Summary Dashboard

This frontend application implements the QE-3198 Monthly Spending Summary Dashboard and associated DAVSpending user stories using AngularJS 1.7.9.

Key features:
- Month selection for spending summary.
- Monthly total spend calculation and KPIs.
- Visual representation of monthly spend via KPI cards and Chart.js breakdown.
- Credit card-only scope enforcement.
- Entry point for deeper spending insights.

Refer to the LLD and Enterprise UI Specification for full design and implementation details.

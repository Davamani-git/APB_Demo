# Monthly Spending Summary Dashboard (DASummary)

This repository contains the AngularJS 1.7.9 Single Page Application implementation of the Monthly Spending Summary Dashboard as per LLD QE-3258.

To run locally:

1. Serve the repository root via a simple HTTP server (e.g., `python -m SimpleHTTPServer 8080`).
2. Navigate to `http://localhost:8080/index.html` in Chrome or Edge.

Mock mode is enabled by default (ENV_CONFIG.useMockData = true). Toggle this flag in `src/app/config/env.config.js` to switch between mock and production API modes.

# Credit Card Expenditure Dashboard

This is a lightweight, front-end only AngularJS application that provides a monthly spending summary dashboard for credit card customers.

## Features

- Summary cards for multiple credit cards
- Monthly spend analysis with bar chart
- Category-wise spend breakdown via doughnut chart
- Top merchants analysis
- Full transaction table with filters and CSV export
- Light/Dark mode toggle

## How to Run

1. Ensure you have an internet connection (CDN dependencies).
2. Create the following folder structure:
   ```
   src/
     index.html
     css/
       styles.css
     js/
       app.js
       controllers/
         dashboardController.js
       services/
         dataService.js
   ```
3. Place the files in their respective locations.
4. Open `src/index.html` in a modern browser.

No build steps or backend services are required.

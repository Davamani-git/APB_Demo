# APB Demo - Monthly Spend Insights

This project implements the Monthly Credit Card Spend Summary Dashboard for INSIGHTS stories.

## Scripts

- `npm install`
- `npm run dev`

Configuration via Vite environment variables:

- `VITE_API_BASE_URL`
- `VITE_USE_MOCK_API` (`true`/`false`)
- `VITE_DEFAULT_CURRENCY`
- `VITE_MIN_SUPPORTED_MONTH`

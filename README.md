# APB_Demo - Monthly Credit Card Spend Insights Dashboard

This project is a React + TypeScript single page application built with Vite. It provides a monthly credit card spend summary dashboard with KPIs, charts, and month selection.

## Scripts

- Install dependencies:

```bash
npm install
```

- Run dev server:

```bash
npm run dev
```

The app will start on http://localhost:5173.

Mock API mode is enabled by default via `VITE_USE_MOCK_API=true` in `.env`.
